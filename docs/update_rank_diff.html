Update the rank differences of the teams.

Usage

update_rank_diff(best_new, best_old)

Arguments

best_new
list of the best submissions per team and per metric as returned by get_best.
best_old
old list of the best submissions per team and per metric.

Value

update_rank_diff returns the input list best_new with an updated column rank_diff for each metric.

Description

Update the rank differences of the teams.
